# Step To Ukraine

## Project Overview

This project implements the server-side component of the Step To Ukraine application. It is built on Nest TypeScript with Swagger to implement the API and Mongoose for interacting with the MongoDB database.

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn run start

# watch development mode
$ yarn run dev

# production mode
$ yarn run start:prod
```

## Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
